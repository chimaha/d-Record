## 特徴

<CardGrid>
    <Card title="Annictに自動送信" icon="rocket">
        アニメの視聴記録ができるWebサービス[Annict](https://annict.com/)に、dアニメストアで見たアニメを自動送信します。
    </Card>
    <Card title="手動で送信" icon="pencil">
        自動送信をオフにして、エピソードごとに手動で送信することもできます。
    </Card>
    <Card title="自動送信のオンオフを切り替え" icon="setting">
        再生ページのアップロードボタンから、作品ごとに送信のオンオフを切り替えることができます。
    </Card>
    <Card title="高い精度でアニメを取得" icon="magnifier">
        dアニメストアとAnnictのアニメ名の表記揺れをカバーします。<br></br>2024年夏アニメでは、92%を正確に取得できています。
    </Card>
</CardGrid>
